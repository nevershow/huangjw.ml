安装Linux双系统教程

Let’s go

下载系统镜像文件

Linux有很多发行版, 这里推荐Ubuntu或Deepin(Deepin是国产的Linux操作系统, Linux下的网易云音乐就出自Deepin, Deepin还内置了搜狗输入法和WPS等应用, 更重要的是内置了QQ2016, 支持远程桌面和视频通话):

  • Ubuntu 16.04 LTS镜像下载
  • Deepin 15.3 镜像下载

制作U盘启动盘

如果你下载的是Deepin镜像, 在iso文件中有一个deepin-boot-maker.exe, 解压出来后可直接运行制作系统盘, 否则进行下面的操作:

  • 准备一个U盘, 右键格式化, 文件系统选择 FAT32
  • 将下载的iso镜像文件直接解压到U盘

注意: 如果待会U盘启动失败, 可下载 软碟通, 使用该软件制作启动盘

分盘

启动盘制作完成需要一定时间, 此时可以进行分盘

  • 右键”计算机”或”我的电脑”,选择”管理”,点击”磁盘管理”
  • 选择一个文件格式为 NTFS 且 可用空间大于40GB 的磁盘
  • 右键”压缩卷”, 输入 40960

(即分配40GB的空间来安装Ubuntu系统, 也可改为其他大小, 建议不要少于40GB, 1GB = 1024MB),点击”确定”,此时会出现40GB绿色的可用空间, 如下图

U盘启动

  • 启动盘制作完成并且完成分盘, U盘不要弹出, 直接关机
  • 确定所用电脑的U盘一键启动按键
  • 按下电源键开机, 开机瞬间 狂按(抢红包)上一步得到的一键启动按键
  • 如果成功一键启动, 会进入 Boot Menu , 选择U盘启动(含有USB, UEFI等字眼的选项)
  • 如果一键启动不成功, 可在开机时进入 BIOS, 不同电脑进入BIOS的按键不太一样, 自行百度. 成功进入BIOS后, 在 Boot 页面将U盘对应选项移到第一, 其他选项不变, 然后保存退出, 如下图

装系统

  • 成功进入U盘启动后, 可以Try(试用)也可以Install(安装), 选择安装
  • 系统语言默认英文, 可以选择中文, 也可在安装成功后随时更改系统语言
  • 安装类型选择”其他选项”, 点击”继续”
  • 点击那个40G的”空闲”盘, 点击左下角的”+”, 确保所有选项选择都与接下来的图一样
  • 分个20G的系统盘
  • 分个2G的交换空间
  • 分个200M的boot盘(记住分出来的boot盘对应dev/sda几, 后面要用到)
  • 将剩下的”空闲”空间全部分成home
  • “安装启动引导器的设备”选择boot盘所对应的dev/sda
  • 确认无误后点击”现在安装”
  • 地方填”beijing”即可, 继续
  • 键盘布局选”汉语”,继续
  • 填写用户信息, 密码建议不要太长, 会经常用到
  • 耐心等待半个小时左右
