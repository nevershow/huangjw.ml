中山大学东校区连接校园网的几种方式

Let’s go

认证客户端

  • Windows
  • Mac OS X
  • Linux 64位

安装后打开客户端，新建一个802.1X连接，输入NetID和密码，选择正确的网卡连接校园网。如果以下客户端配置正确仍无法上网，欢迎致电网络与信息技术中心: 84036866。

各系统安装客户端注意事项:

  1. Windows
    直接选择默认安装即可，也可修改路径，但路径中不能有中文，在连接的属性中网卡选择带有Family Controller或者Ethernet字眼的网卡。

  2. Mac OS X
    苹果电脑在开机前先插上USB转换器，开机后点击左上角的苹果图标->关于本机->系统报告->以太网卡，查看以太网卡是en几，然后在launchpad中打开客户端，连接属性中选择对应的网卡，一般是大于en2的，也可以每个都尝试一下登录。如果上不了网，进入系统偏好设置，找到当前连接的以太网，点开高级，配置IPV4选择使用DHCP，取消勾选802.1X中系统自带的连接。

  3. Linux
    Linux下安装客户端比较复杂，这里给出教程：

  • 打开下载的文件所在文件夹, 右键提取到此处(Extract Here)
  • Ctrl + ALT + T 打开终端，cd到当前文件夹
  • sudo mkdir /usr/iNode，如果已存在iNode文件夹无需创建或者可以删了新建
  • sudo cp -r iNodeClient /usr/iNode/，把文件夹移到usr/iNode下
  • cd /usr/iNode/iNodeClient，切换目录
  • sudo sh install.sh，进行安装
  • sh iNodeClient.sh，运行客户端，可以锁定到启动器方便以后打开
  • 如果出现中文乱码，可输入sudo echo ‘export LANG=zh_CN.UTF8’ >> /etc/profile，英文引号
  • 输入NetID和密码, 网卡选中eth0或en开头

YaH3C

Linux或Mac安装不上iNode客户端备选方案，请看这里>>>，版本较新的Linux安装后运行可能会报错无法使用，最新版的苹果系统，客户端检测不到USB网卡无法上网，可尝试YaH3C，在设置网卡时默认是eth0，需要改为对应的en几。

极路由

极路由配置非常简单，插上电源后过一会儿会出现一个HiWifixxxx，后面的数字是路由器背面Mac地址的后几位，连接该wifi后输入路由器背面的网址进入配置页面，跳过设置上网方式，选择华三认证或H3C认证

根据下图进行配置(加密方式标准或者西南大学都可尝试一下)

虽然学校不反对使用路由器，但也不支持学生使用路由器，如果因为使用路由器出现网络问题请自行解决。
